# MyWeatherApp

MyWeatherApp is a weather application that fetches and displays weather data for various locations. This project is built using TypeScript and Node.js, providing a robust structure for managing weather-related data.

## Features

- Fetch current weather data from a weather API.
- Display weather information including temperature, humidity, and conditions.
- User-friendly interface for inputting location data.

## Installation

To get started with MyWeatherApp, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/MyWeatherApp.git
   ```

2. Navigate to the project directory:
   ```
   cd MyWeatherApp
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application, use the following command:
```
npm start
```

This will start the application and you can access it in your browser.

## Development

For development purposes, you can run:
```
npm run dev
```

This will start the application in development mode, allowing for hot reloading and easier debugging.
